# Циклы 

### Цикл **for**  

for(*задание переменной*; *условие*; *действие*){  
    /* тело цикла */  
}  
```javascript
/* вывод в консоль i от 0 до 9 */
for(let i = 0; i < 10; i++){
    console.log(i);
}   
```

### Цикл **while**  

while(*условие*){  
    /* тело цикла */  
}  
```javascript
/* вывод в консоль i от 0 до 9 */
let i = 0;
while(i < 10){  
    console.log(i);
    i++;
}   
```
