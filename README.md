### Основы для понимания структуры курса
1.  1 папка = 1 занятие  
2.  Виды коментариев:  
**javascript:**  
```javascript
  /* !важные */  
  /* пояснения */  
  /* -скрытый код */  
```  
**html:**  
```html
  <!-- !важные -->  
  <!-- пояснения -->  
  <!-- -скрытый код -->  
```  
**css:**  
```css
  /* !важные */  
  /* пояснения */  
  /* -скрытый код */  
```
