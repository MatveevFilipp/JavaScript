### Основы для понимания структуры курса
#### Виды коментариев:  
##### javascript:  
```javascript
  /* !важные */  
  /* пояснения */  
  /* -скрытый код */  
```
##### html:
```html
  <!-- !важные -->  
  <!-- пояснения -->  
  <!-- -скрытый код -->  
```
##### css:
```css
  /* !важные */  
  /* пояснения */  
  /* -скрытый код */  
```
